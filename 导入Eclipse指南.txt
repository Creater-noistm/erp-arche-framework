 始祖ERP框架 — Eclipse 导入指南
══════════════════════════════════════════════════════════════

你的 Eclipse 路径：
  D:\Downloads\eclipse-java-2026-03-R-win32-x86_64\eclipse\eclipse.exe

──────────────────────────────────────────────────────────
 方式一：【推荐】一步导入（已配好 .project 文件）
──────────────────────────────────────────────────────────

步骤 1：双击启动 Eclipse（可以双击桌面的快捷方式）

步骤 2：Eclipse 启动后，关闭欢迎页（Welcome）

步骤 3：菜单栏选择：
   File → Import...
   
步骤 4：在弹出的窗口中选择：
   Maven → Existing Maven Projects  →  Next
   
步骤 5：在 Root Directory 后面点 Browse...，选择：
   D:\2026\Java大作业\erp-framework

步骤 6：确认 pom.xml 被勾选 → 点 Finish

等待 Eclipse 下载依赖（右下角进度条跑完就OK了）

──────────────────────────────────────────────────────────
 方式二：双击 .bat 脚本自动打开（需先运行一次方式一）
──────────────────────────────────────────────────────────

第一次按方式一导入后，以后可以直接双击桌面上的：

   「启动ERP框架_Eclipse.bat」

这个脚本会自动打开 Eclipse 并定位到工作区。

──────────────────────────────────────────────────────────
 导入成功后的操作
──────────────────────────────────────────────────────────

1. 在左侧 Package Explorer 展开 erp-framework
2. 找到 src/main/java → com.erp → ErpLauncher.java
3. 右键 → Run As → Java Application
4. 即可看到始祖ERP框架的主窗口！

注意事项：
  • 首次导入 Eclipse 会下载 Maven 依赖（约2-5分钟）
  • 确保 JDK 17 已配置：Window → Preferences → Java → Installed JREs
  • 如果代码报错，右键项目 → Maven → Update Project (Alt+F5)
